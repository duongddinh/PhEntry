Forget password-bruteforcing method

To use this, you need a list of possible passwords that you might use for your login.
Write all possible passwords and store it in a txt file.
Then select the txt files for the program to start checking.

Uses SHA265 algorithm


    	public static String getHash(String pssd) {
		MessageDigest md = null;
		try {
			md = MessageDigest.getInstance("SHA-256");
		} catch (NoSuchAlgorithmException e) {
			e.printStackTrace();
			wl.writeLog(e.toString());

		}
		byte[] digest = md.digest(pssd.getBytes(StandardCharsets.UTF_8));

		return DatatypeConverter.printHexBinary(digest).toLowerCase();

	}

Note: bruteforcing in some cases are illegal to perform.
Since this program demonstrates over-engineering theme, I added this with no malicious intend.

Hide PopUp

Hide all will disable method call for EssentialFunctions.popUp(str, str).
Otherwise, method call EssentialFunctions.PopUp(str,str) will be called depends on var bool.
You won't be able to hide start-up popUp because they are important.

Hide recommeded popUp: hide what we think we should hide
Hide "Not show password" recommendation popUp: Hide annoying popUp when you did not choose "Not show password"
(Showing password may potentially get your password stolen\ncheck the option if you don't want to take a risk!)
Hide not recommended popUp: Basically an inverse version of "Hide recommeded popUp."


    	protected static void popUp(String todis, String title) {
		if (!HidePopUpOPtion.hideAllP) {
			JOptionPane.showMessageDialog((Component) null, todis,
					title, JOptionPane.INFORMATION_MESSAGE);
		}
	}

import java.awt.Graphics;
import java.awt.Image;
import java.awt.Toolkit;

import javax.swing.JPanel;

public class MainScreen extends JPanel{
	private static final long serialVersionUID = 1L;
	public void paint(Graphics g) {
		drawDabg(g);
		// can draw more here!
	}
	// Basically this draws the gif you see in the home screen
	private void drawDabg(Graphics g) {
		Image icon = Toolkit.getDefaultToolkit().getImage(getClass().getResource("Nice1.gif"));
		g.drawImage(icon, 200, 100, this);
	}
}

About Loging in

You can only create 1 account only, if you create another, you need to detele your pre-existing one.

When you log in, you will see the window saying welcome, [yourname] and reddit.
This is the webview version of the website which can be accessed here.
You are NOT actually logged in to reddit but, rather, just got to view it.
Your login and password are not stored on the server;
it is just a simple javascript to show yor username you created on the server.
